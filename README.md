# LrcEdit-Android
安卓版的歌词编辑器
[项目介绍](https://www.licz.site/2018/04/04/%E9%A1%B9%E7%9B%AE%E4%BB%8B%E7%BB%8D/)

已实现功能：
* 添加歌词
* 保存为TXT文件(根目录下test.txt文件)
* 丑陋的界面
* 较差的用户体验

待实现的功能：
* 添加歌词时间
* 保存为lrc文件格式
* 自定义文件名
* 修改已添加的歌词
* 预览歌词
* 精美的界面
* 良好的用户体验
